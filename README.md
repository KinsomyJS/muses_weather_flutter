### Muses是一款用Flutter编写的天气预报应用，代码会持续更新...

使用api为魅族天气api，感谢 [https://github.com/jokermonn/-Api]()

### 功能
* 支持全国地级市天气预报
* 未来五天天气预报查看
* 支持生活指数查看
* 支持自定义更换背景
* 保存城市天气列表
